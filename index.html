主页会标出掌握度低于 60% 的「薄弱词」并提醒你；在「学习」页点「⚠️ 薄弱词巩固 › 开始巩固」可把这些词单独拉出来练。
🔄 练习页上方换方式›
进入任意练习后，顶部有一排「🃏 闪卡 / ✅ 选择题 / ✍️ 拼写 / 🔊 听音」按钮，练着练着可随时换一种方式，不用退出重进（今日新词学习仍保持四种方式各自独立通关）。
➕ 添加页›
加新词时如果词库里已有，会弹出提示“你 X 天前背过啦”，不会重复添加；拼写很像的词（如 comunicate 和 communicate）也会提醒你。
📚 词库页›
随时搜索、查看每个词的复习历史和掌握度。
🔊 发音›
点 🔊 按钮听标准英语发音，单词和例句都能读。电脑 / 带英文嗓音的设备会用系统朗读（可离线）；手机 App 会自动改用在线音频发音（需联网），多个音源自动切换，联网即可正常出声。可在下方“🔊 发音设置”里切换美式 / 英式口音。
🌐 联网查词 / 翻译›
在“词库”页顶部输入单词点“查词”，会联网给出音标和中文翻译。添加新词时也可以先“联网查一下”让系统自动帮你填好中文释义和音标。
🔒 数据安全›
所有单词只存在你这台设备的浏览器里。想换设备？用“设置”里的导出/导入把词库带走。
⛙️ 设置 & 数据›
导入时相同单词会自动合并，并保留你更早的“首次背词时间”。

☁️ 云备份（联网核查）

✅ 本应用已由管理员统一接入云端。打开后填一次真实姓名即可（无需注册、无需邮箱/手机号），之后学习进度自动同步到看板。
数据存到你自己的免费 Supabase 云库；家长在电脑打开「监控看板.html」即可看到所有孩子的学习进度、打卡与正确率。首次使用请看随包的《云端看板配置指南》。
说明：服务器是你自己在家跑的小程序（见备份包里的 backup-server.js），数据只存在你这边，不会上传给任何公司。手机和电脑用同一个“设备名”就能互相备份恢复。

🔄 离线学习与冲突解决

没网时学的进度只存在本机，联网后会自动核查云端：

  • 只有本机或只有云端有变化 → 自动同步，不打扰你；
  • 两边都有变化 → 弹框让你手动二选一：📤 以本机为准（上传覆盖云端）/ 📥 以云端为准（下载覆盖本机）/ 稍后再说；
  • 没网期间的操作，联网后都按你的选择统一，不会互相覆盖丢失。
